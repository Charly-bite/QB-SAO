# ⚡ static/js/alpine.min.js

> **Alpine.js** — Lightweight reactivity framework.

## Role
Provides declarative reactivity via `x-data`, `x-show`, `x-on`, etc. directives in HTML templates. Used for interactive UI elements like modals, tabs, dropdowns, and dynamic forms.
